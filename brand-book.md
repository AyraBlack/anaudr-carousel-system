# anaudr brand book

## Purpose

This document defines the anaudr brand identity for Instagram carousels and related visual content.

Its job is to make sure all carousels feel like they come from the same brand, not from seven different personalities with Wi-Fi.

## Brand objective

The anaudr brand should feel:
- recognizable
- consistent
- sharp
- intentional
- premium
- easy to read
- emotionally clear

## Brand foundations

### Brand name
anaudr

### Main use case
Instagram HTML carousels

### Main goals
- build recognizability
- create consistency
- speed up content production
- maintain a strong brand voice
- reduce random design decisions
